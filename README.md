RapidReport/barcode-java
==================
RapidReport/barcode-java は Java 用のバーコードのレンダリングエンジンです。

対応規格
-------
EAN8（JAN8）

EAN13（JAN13）

CODE39

CODE128

Codabar（NW-7）

動作環境
-------
Java6 以上（Eclipse プロジェクト）

ライセンス
-------

改変・再配布は自由です。

利用した場合の報告は必要ありません。
